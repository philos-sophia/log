经济学原理：生产与供给曲线

11 / 21 / 2021 / 日 / 张维迎

生产是指从投入到产出的转换，生产函数是要素投入量与产品产出量之间的数量关系。

在短期内，只有一种或部分要素的投入可变；在产量超过一定水平后，可变要素的边际生产率会递减。但长期来看，所有要素的投入量都可以改变，生产函数可以表现出规模报酬递增、不变、递减三种情况。

当所有要素投入可变时，企业选择最优要素组合使得生产成本达到最小。成本最小化的条件是：要素在生产上的边际替代率等于要素价格之比；或者说，最后一元钱的成本支出无论用于哪种要素，对产出的边际贡献相等。因此，成本最小化要素组合随要素价格的变化而变化。

企业选择最优的要素组合相当于选择某种特定的生产技术。短期成本不可能低于长期成本。

边际成本曲线与平均成本曲线在平均成本的最低点相交。

最优产量在边际收益等于边际成本时达到。在完全竞争的市场上，对单个生产者而言，边际收益等于产品的价格。

短期供给曲线是平均可变成本最低点以上的边际成本曲线。长期供给曲线是长期平均成本最低点以上的长期边际成本曲线。无论短期还是长期，供给量随价格的上升而上升。

在没有法定准入限制的情况下，一个企业想持续赚取利润，就必须通过不断创新，获得别的企业难以获得的技术和资源。

1. 生产与供给曲线

1.1. 生产技术

生产过程是指从投入到产出的转换，其中投入的物品被称为要素投入，产出的物品被称作产品。

1.1.1. 生产函数

生产函数: 投入与产出之间的关系。

y = f(x1, x2, ..., xn)
y 代表某种产品的产量，xi代表第i种要素的投入量。

生产函数通常被认为是技术性的，但这种技术关系不仅与物质形态的技术有关，还受到企业内部管理制度和外部环境的影响。

短期: 只有一种或部分要素投入可变的时间跨度。

长期: 所有要素投入都可以改变的时间跨度。

1.1.2. 短期生产函数

单一要素投入的总数量称为投入总量，单一产品的产出总量称为总产量，分别为我们前面提到的x和y。总产量除以投入总量称为要素的平均产量，记为 AP = y/x。每增加一单位的要素投入所增加的产出量称为要素的边际产量，记为 MP = dy/dx。

边际生产率递减法则：在其他要素投入不变、只有一种要素投入可变的情况下，可变要素投入超过一定的临界值后，其对应的边际产量随着投入的增加而下降。

事实上，平均产量下降是边际产量下降的结果，这是因为新投入带来的产出数量低于原来的平均值，因此新的平均值被拉低了。

边际生产率递减法则适用于所有的要素投入，包括劳动、资本、士地等。中国实施城镇化战略，大量农民离开土地进入城市，士地上工作的劳动力数量大大减少，但粮食产量并没有减少，一个重要原因是原来的农业生产中，劳动力的边际生产率为零．甚至是负的。

知识拓展: 马尔萨斯陷阱

两百多年前，英国经济学家马尔萨斯提出了一个著名的观点：由于边际生产率递减法则的作用，随着人口增多，人均粮食产量会越来越少，人类永远只能生活在生存线上，生活水平不可能提高。这被称为马尔萨斯陷阱。

马尔萨斯的理论基本上造用于马尔萨斯之前的时代，但不适用于马尔萨斯之后的时代。

为什么马尔萨斯的悲观预测没有出现？因为经济不仅受边际生产率递减法则的支配，也受技术进步的支配。如果技术进步速度足够快，边际生产率递减法则就不会表现为人均产出的减少，因为技术进步可以提高边际生产率。

1.1.3. 长期生产函数

长期来讲，所有要素投入都是可以改变的，企业可以通过调整不同要素的组合来实现所希望的产出。经济学家特别关注两种生产要素：资本和劳动。

在只有资本和劳动两种要素的情况下，生产函数变为：

y = f(K, L)

这里K代表资本，L代表劳动。

资本在物质形态上表现为厂房、机器设备、中间品、原材料等。但经济学上讲的资本一般指货币（价值）形态的资本，其原因在于，上述物质形态的资本都可以在市场上用货币形态资本购得或租赁得到。

劳动指劳动者的体力、知识、智力的投入。劳动必须与资本结合才能生产出产品，否则就成了 “巧妇难为无米之欢”。

当然，从人类历史的发展来看，财富最初都来白于劳动，随着社会的发展，劳动所得积累成了资本，如我们在鲁滨孙经济中看到的。

一般来说，要素投入在生产上具有可替代性。就是说，同样的产出数量，可以由不同的要素组合来生产。比如，可以多用资本、少用劳动，也可以少用资本、多用劳动。

等产量曲线（isoquant），表示某一固定产出下对应的所有投入组合的连线。

无差异曲线给出的是相同的效用水平，等产量曲线给出的是相同的产出水平。资本与劳动的可替代程度取决于等产量曲线的形状，是由技术决定的，与要素价格无关。

边际技术替代率: 在产量不变的前提下，增加一单位某种要素投入可以替代另外一种要素的数量。

边际技术替代率递减法则: 随着一种要素投入的增加，该要素可替代其他要素的数量不断递减。

1.1.4. 规模报酬

当所有投入增加一倍时，产出也增加一倍，则称该生产技术为规模报酬不变的；当所有投入增一加倍时，产出增加小于一倍，则称该生产技术为规模报酬递减的；当所有投入增一加倍时，产出增加大于一倍，则称该生产技术为规模报酬递增的

一般来说，规模较小时，增加投入有规模报酬递增；但当规模大到一定程度时，就会出现规模报酬递减。

1.2. 生产成本

1.2.1. 成本函数和成本曲线

生产成本是指企业为要素投入支付的代价．它不仅依赖于要素投入的数量，而且依赖于要素的价格。经济学上，产量与为生产该产量而必须支付的最低成本之 间的关系，被称为成本西数，记为 C = C (y)，其中y表示产量。在图形上，成本函数就变为成本曲线。

根据生产上的要素投入的可调整性，我们需要区别短期成本函数和长期成本函数以及对应的短期成本曲线和长期成本曲线。

短期成本是指只有一种要素投入或部分要素投入可变时的最低成本，长期成本是指所有要素投入都可变时的最低成本。

1.2.2. 短期成本曲线

在短期内，由于有一部分要素投入的数量给定的，这些要素投入对应的成本不随产量而变化，因而被称为固定成本（FC)。相应的，可变要素的投入量随着产量的改变而变化，对应的成本被称为可变成本(VC)。固定成本和可变成本之和构成了总成本(TC)。

固定成本: 不随产量变化的成本：等于固定生产要素的支出。

可变成本: 随产量增减而变化的成本：等干可变生产要素的支出。

固定成本和可变成本的相对比例在不同产品中是不同的。一般来说，资本密集型产品的固定成本比重较大，而劳动密集型产品的可变成本比重较大。这是因为．机器设备、厂房等固定资产都是固定成本，而劳动成本、原材料投入是可变成本。当然，如果企业没有解雇工人的自由，劳动成本也就成为固定成本。

注意，根据定义，固定成本和可变成本的区别只对短期有意义。从长期看，因为所有要素投入都是可变的，所以所有成本都是可变成本。

平均成本: 总成本与总产量之比，记作 AC = TC/y。

平均固定成本: 固定本与总产量之比，记作 AFC = FC/y。

平均可变成本: 可变本与总产量之比，记作 AVC = VC/y。

平均成本和边际成本之间的关系是：当平均成本高于边际成本时，平均成本随着产量的增加而降低；当平均成本低于边际成本时，平均成本随着产量的增加而上升。

1.2.3. 长期成本曲线

生产函数: y = f(K, L)，其中K和L分别表示资本投入和劳动投入，

成本函数: C = Kr + Lw，其中K和L分别表示资本投入和劳动投入，r和w分别表示资本和劳动力的价格。

分析长期成本函数有两个步骤：第一步，给定产量目标，企业如何选择成本最低的要素投入组合；第二步．最低成本如何随产量而变化。

等成本曲线的斜率(绝对值)是劳动价格和资本价格的比例。

同等产量曲线类似于消费者理论中的无差异曲线一样，等成本曲线在形式上类似于消费者理论中的预算约束线。不同的是，在消费者行为理论中，消费者要解决的问题是如何在给定的预算约束线下，通过选择产品组合实现效用最大化；而在这里，企业面临的问题是如何在给定的产量下，通过选择投入组合实现成本最小化。

从等成本曲线和等产量曲线的图中可以看出，最优组合点满足这样的条件：等产量曲线的要素边际替代率（绝对值）等于要素价格比率。

换言之，最优组合要求最后一单位的成本支出不论花在资本投入上还是劳动投入上，其边际产量是相同的。如果不是这样，企业就可以通过调整投入的比例降低成本。比如说，如果对于最后一单位成本支出来说，资本的边际生产率大于劳动的边际生产率，通过用资本替代劳动就可以降低成本支出。

规模经济: 成本增长幅度等于或小于产量增长幅度。

长期成本函数和长期成本曲线的具体形状，依赖于生产函数的特征和要素价格如何随产量变化。如果成本的增长慢于产量的增长，则长期成本西数是递增的山函数；如果成本和产量按照固定的比例增加，则长期成本函数是递增的线性函数；如果成本的增长快于产量的增长，则长期成本函数是递增的凸函数。在经济学上，前两种情况统称为规模经济，第三种情况称为规模不经济。

值得一提的是，规模经济与规模报酬是两个相关但并不完全相同的概念。一个明显的特征是：规模经济的概念涉及成本问题，而规模报酬递增是纯技术问题。规模报酬递增是形成规模经济的一种主要因素，但既不是必要因素也不是充分因素。即使技术上生产函数为规模报酬非递增的，但如果劳动力价格w、资本价格r随着产出增加而下降．那么成本上升的速度仍然可能低于产量增长的速度，从而形成规模经济；反之，即使技术上生产函数为规模报酬递增的，但如果劳动力价格w、资本价格r随着产出增加而递增．那么成本上升的速度仍然可能高于产量增长的速度，从而形成规模不经济。当然，如果要素价格与产量无关，像我们一直假定的那样，则规模经济和规模报酬概念是对应的。

长期平均成本（LAC）: 长期成本与总产量之比。

长期边际成本（LMC）: 增加一单位总产量导致的长期成本的增加值。

长期平均成本曲线和长期边际成本曲线相交于长期平均成本的最低点。

1.2.4. 长期平均成本于短期平均成本的关系

长期中，对于任何计划生产的产量y，企业总是选择成本最小的要素投入组合，这也是选择平均成本最小的要素投入组合；短期内，企业只有在生产该计划产量时，短期平均成本才是最低的。

1.3. 利润最大化

1.3.1. 企业的目标

首先，利润这个目标是从私有制企业推出来的。在私有企业中，所有者最关心的当然是利润。而在现实情况中，并非所有企业都是私有企业，还存在很多国有企业，后者通常并不以利润最大化为目标。其次，即使是私有股份制企业，也普遍存在着所有权与控制权分离的现象，这将产生委托——代理问题。有可能出现决策者追求做大规模、管理更多的人、扩大自己的权限等符合代理人自己的利益而不符合委托人目标最大化的决策。最后，利润有长期和短期之别，真正的企业家会注重长期利润，宁愿短期亏损也不愿意损害企业的长期利益。因此，企业可能也不是以当前利润最大化为目标的。此外，还有一些社会性因素约束利润目标。当下流行着一个很时髦的词叫作 “企业社会责任”，要求企业要为消费者和社会公众考虑。如果企业利润与社会责任存在冲突，企业有时候也不得不放弃利润最大化的目标。传统的经济学通常不考虑企业目标与社会责任之间存在的矛盾，但在非经济学领域，很多学者却在论证、研究二者之间的矛盾。

尽管利润最大化目标的假设存在上述缺陷，但在市场经济中，它仍然是我们在理论上可以接受的个假设。因为，从长期看，如果一个企业持续不能赚钱，它是没有办法生存下去的。并且，企业决策者其他目标的实现，也在很大程度上依赖于利润。因此，在本书中，我们仍然遵循传统经济学的做法，假定企业的最终目标是利润最大化。

会计上，企业的利润可以表示为销售收入和成本之差。销售收人可以表示为价格与产量的乘积，成本是为生产该产量而必须支付的要素投入的支出。

如果我们用π表示利润，R表示收入，C表示成本，y表示产量，我们有如下等式：π（y） = R（y） - C（y）

知识拓展: 经济利润与会计利润

会计利润和经济利润的区别在于二者对成本的定义不同。经济学中的成本是为生产而付出的所有机会成本，而会计账面成本不包括隐形的机会成本。

机会成本本质上不是成本，而是放弃的收益，即将资源投入当前生产所放弃的其他机会最大可能的收益。

有些机会成本是无法从账目中观察到但却对决策者有影响的隐性成本。比如，很多国有企业占用大量的土地以及很多其他资源，这些资源本来可以通过市场拍卖，产生相应的收益，从经济学的视角看是一笔巨额的机会成本。这些资源免费投入国企中，但对应的机会成本没有计算在企业财务报表之内。因此，会计上盈利的国有企业在经济上可能是亏损的。

沉默成本(sunk cost):

与机会成本相对应的另一种成本是沉没成本。沉没成本是指已经支付的、无法挽回的成本。

经济成本 = 会计成本 + 隐形机会成本 - 沉没成本

经济利润 = 会计利润 - 隐形机会成本 + 沉没成本

值得一提的是，在短期内经济学上通常把固定成本看作沉没成本。这种做法在有些情况下并不成立。例如，购买到的机械设备虽然在短期内无法转让，属于固定成本，但是它并不是沉没成本。因为如果停止生产，机械设备对应的成本至少是部分可以变现的。再如，很多专用性芯片购买后，其对应的成本即变成了沉没成本，但它们通常是按比例与产品匹配生产，因而应该被视作可变成本。

1.3.2. 利润最大化条件

利润函数: π= π（y）

利润曲线: 利润函数在产量——利润坐标图上，就是利润曲线。

利润曲线的变化轨迹总是总收入曲线和总成本曲线共同的作用的结果。

总收入曲线与总成本曲线距离最大之点即利润最大之点，对应的产量就是最优产量

从总成本曲线和总收入曲线的图中还可以看出，最大利润在总收入曲线的斜率等于总成本曲线的斜率时达到。总收入曲线的斜率是经济学上定义的产量的边际收益，即增加一单位产量所带来的收入的增加；总成本曲线的斜率是边际成本。由此，我们得到利润最大化产量的基本条件：边际收益 = 边际成本。

边际收益: 产量增加一单位带来的销售收入的增加额。

在完全竞争市场，利润的最大化产量的条件是，价格等于边际成本：p = MC（y）。

1.3.3. 短期利润与生产决策

当价格大于平均总成本时，生产可以带来利润。但如果短期利润小于零。企业是否应该选择停产呢？答案取决于市场价格与平均可变成本的比较。

迷你案例: 为什么航空公司会提供超低票价

在飞行过程中，燃油费、折旧费等固定成本项构成了成本的主要部分，而多一个乘客带来的边际成本却接近于零。对于航空公司而言，重要的不是固定成本，而是边际成本。这是因为，不管飞机上是坐满了人还是空无一人，只要飞机一起飞，就必须支付相应的固定成本，但只要能以高手边际成本的价格多卖出一张票，他们就能多賺一份钱（或者少亏一份钱)。

1.3.4. 长期利润与生产决策

如果没有进入壁垒，如果所有的技术和生产要素对所有的企业都可以以同等的条件（价格）获得，企业不可能持续地赚取利润。这是因为，利润的存在会诱使新企业的进入，竞争会降低市场价格，直到价格等于平均成本为止。因此，长期均衡结果一定是市场价格处于与长期平均成本最低点相等的水平。此时，D = MR = LAC = LMC，利润为零。

长期均衡: 价格等于最低长期平均成本。

创新意味着长期均衡是不存在的。对一个企业来说，最重要的并非我们在前面分析的给定技术和产品情况下如何选择要素组合和产量，而是创造新的技术和产品。

1.4. 供给曲线

1.4.1. 单个企业的供给曲线

消费者的选择决定了需求函数和需求曲线，生产者的选择决定了供给函数和供给曲线。

供给函数或供给曲线，是指在不同价格水平下企业愿意生产的产量，记为 y = S (p)。

短期供给曲线: 短期平均可变成本最低点以上的边际成本曲线部分。

长期供给曲线: 长期平均成本最低点以上的长期边际成本曲线部分。

1.4.2. 供给价格弹性

与消费者理论中用需求弹性度量需求对价格的敏感度类似，在生产理论中，我们用供给价格弹性度量企业产量对价格的敏感度。

供给价格弹性: 供给量变化率与价格变化率的比值。

根据我们前面的描述，生产过程分为短期和长期。短期内企业调整产量的自由度较小，但是随着时间的推移，企业可以更加灵活地改变生产方案。因此短期供给弹性小于长期供给弹性。比如说，假如粮食价格涨了一倍，粮食的供给量不会马上发生巨大的变化。但是一年之后，粮食的供给量就会出现明显的增长。

1.4.3. 市场供给曲线

市场供给曲线又称产业供给曲线，在产量——价格坐标图上，它是又所有生产企业的供给曲线横向加总得到的。

在一个市场上，商品的价格上涨通常会引发市场总供给的增长。这种增长来自两个方面：第一，当商品价格上涨时，原有生产企业根据各自的供给曲线增加自己的产量。 第二，商品价格上涨可能诱使生产者的数量增加。在原来的价格下，有些企业不会生产该商品，但当价格上涨到超过其最低的平均可变成本时，它们就会开始选择生产该商品。一般来说，一种产品的价格越高，生产该产品的企业的数量越多。

1.4.4. 供给曲线的移动

从长期来看，供给曲线的移动主要是由技术变化、其他产品价格变动以及投入品价格变动等因素导致的边际成本的变化引起的。

如果同样的投入品用于A产品的生产，就不能用于B产品的生产，或者种产品产量的增加会提高另一种产品生产的边际成本，那么，A产品和B产品就是竞争品。比如，企业内同样的人力可以用于计算机的生产，也可以用于手机的生产，则计算机和手机就是竞争品。因此，如果手机价格上升，生产计算机的机会成本就会上升，意味着在计算机价格不变的情况下，企业会减少计算机的产量。

如果同样的要素投入可以同时生产两种产品，或者一种产品产量的增加可以降低另一种产品生产的边际成本，则这两种产品就是生产上的互补品。比如，牛肉和牛皮就是典型的互补品。当牛肉的价格上升时，农场主就会增加牛肉的供应。此时，即使牛皮的价格不变，牛皮的产量也会增加，也就是说，牛皮的供给曲线会向右移动。

尽管有多种因素在推动供给曲线的移动，但从历史上看，技术进步导致的成本下降，是供给曲线向右移动的主要原因。比如，20世纪90年代早期，个砖头大的手机（俗称“大哥大”）售价在3万元人民币以上，而现在一个普通手机的售价只有几百元，而其功能是 “大哥大” 手机的数百倍。这就是技术进步导致的价格下跌。

1.4.5. 从机会成本看供给曲线

事实上，所有的生产成本都是机会成本或机会成本的某种变形。
