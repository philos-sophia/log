# 训练的四个变量: 训练量 强度 恢复 频率
# 轻松跑本质上是舒适并可以进行聊天的奔跑。
轻松跑E:
  周跑量: 25% 150分钟 32公里
配速跑M:
  周跑量: 20% 110分钟 29公里
门槛跑T:
  周跑量: 10% 
  节奏跑: 20分钟
  巡航间歇:
    - (1.6KM w/1分钟)
    - (3.2KM w/2分钟)
    - 训练-恢复时间比大约为5:1
间歇跑I:
  周跑量: 08% 10公里
  艰苦跑: (6x3分钟H w/2分钟jg)
重复跑R:
  周跑量: 05% 08公里
    - 训练-恢复时间比大约为1:2~3
跨步跑ST:
  - 持续时间短的轻量快速跑，在两次跨步跑间有充分的休息时间
  - 有助于改善跑步效率，可以让你在体能提高后进行更快速的奔跑做好准备
  - 可以将跨步跑和E跑结合起来进行
  - 将其作为进行单调定速跑的调节方式
  - 你或许就会在E跑时跑到那些路面比较平坦柔软的地方去
