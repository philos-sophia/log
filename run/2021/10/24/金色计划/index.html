# 跑步时间 总时间 距离

# E代表轻松跑，L代表轻松的长距离定速跑
# 跨步跑（ST）是轻量的20秒快速跑（非冲刺跑），每两次跨步跑间有60秒的休息
# R代表重复跑，配速为你能在比赛中坚持跑上5分钟的配速
# 艰苦跑（H）本质上是间歇，配速为你能在比赛中坚持10～15分钟的配速
# T代表门槛跑，强度为舒适的艰苦
# 慢跑（jg）则应该以轻松配速进行
#
# 每周有6次训练
# 每周的第4天为机动训练日
# 周日作为每周的第一天
# 比赛前安排2个或3个轻松日的训练
# 比赛后，每3000米比赛距离就安排一个轻松日
