E: # Easy running
  MHR: 65%~78%
  VdotO2max: 59%~74%
  Period: 30~150m
  Weekly: 25%~30%
  Pros: 
    - 加强心肌
    - 抵御伤病
    - 改善血液输送
    - 增强自信
  # 通常不会总在盼望着训练快点结束
L: # Long slow distance
  - Easy running
  - Max(weekly 25%, 150m)

M: # Marathon-pace running
  MHR:
  VdotO2max: 75%~84%
  Period: Min(40~110m, 29KM)
  Weekly: 15%~20%
  Pros:
    - 训练配速

T: # Threshold running
  MHR: 65%~78%
  VdotO2max: 83%~88%
  Period: 5~20m; 5:1W/R
  Weekly: 15%~20%
  Pros:
    - 耐力
    - 清除乳酸
  # 你确实希望它快点结束
  # 但却还能继续跑上好几公里
  # 舒适的艰苦，不是艰苦
  # 节奏跑 巡航间歇

I: # Interval training
  MHR: 65%~78%
  VdotO2max: 95%~100%
  Period: 5m; 1:1W/R
  Weekly: Min(10KM, Weekly 8%)
  Pros:
    - 有氧能力
H: # Hard running
    # I 训练也可以以H跑的方式进行，
    # 即艰苦跑，
    # 不规定要在多少时间内跑多少距离。
    # 例如，6x3分钟H w/2分钟jg。

R: # Repetition training
  MHR: 65%~78%
  VdotO2max: 105%~120%
  Period: 2m; 1:2~3W/R
  Weekly: Min(8KM, Weekly 5%)
  Pros:
    - 无氧能力
    - 效率
    - 速度

# 赛季训练4阶段
BFIP: # 1
  # Base Foundation 
  # and Injury-Prevention 
  # 以打基础和防止受伤为重点的基础训练
  # E跑 + L跑(25%~30%)
  #  1  2  3
  # 13 21 23
IQ: # 4
  # Initial Quality
  # 初级的高质量的训练
  # R跑 + L跑
  #  7  8  9
  # 18 19 20
TQ: # 3
  # Transition Quality
  # 过渡性高质量训练
  # 4阶段中最费力的阶段
  # I跑 + R跑 + L跑
  # 10 11 12
  # 14 15 16
FQ: # 2
  # Final Quality
  # 旨在让跑者达到最佳表现
  # T跑 + R跑 + L跑
  #  4  5  6
  # 17 22 24

# 训练的6秒规则
# T配速要比I配速每400米慢6秒
# I配速要比R配速每400米慢6秒

L配速:
  # 舒适，保守
R配速:
  # 跑完1.6公里的保守估计的配速
I配速:
  # R配速，每400米减慢6~8秒
