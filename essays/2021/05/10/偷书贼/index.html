偷书贼

05 / 10 / 2021 / 月

从5月5号中午12点，到5月10号晚上9点，断断续续终于把《偷书贼》读完。相较于之前阅读的《理想国》和《哲学的故事》以及《鼠疫》，这本中篇小说读起来算是非常的“轻松”。这里之所以给“轻松”打上引号，是因为这里的轻松是只思考的压力，或者对文字的理解没有读哲学相关的晦涩，而打上引号的含义是指马库斯的《偷书贼》的主题是以二战期间德国的一个小镇为背景，主灰色调地讲述了死神、偷书贼和战争的故事。

前天的时候，因为工作开会的需要，我下午去了位于外滩的BFC金融中心的办公室，并顺路领取了外接的大屏显示器。去年的时候，陪伴我驻扎在客户现场的 2014 款的MacBook Pro，可能是太久不曾归家的缘故，在我一次回家后，内置显示器出现雪花闪烁，时好时坏，并最终闪瞎黑屏，彻底歇菜。让人不得理解的是，这个内置屏幕却可以用 Windows 系统点亮（尽管依然有轻微的雪花）。于是乎，昨天上午，借助这台从BFC金融中心带来的外接显示屏，我将苹果电脑的双系统的 Windows 删除，并将 macOS 系统升级至最新版本 macOS Big Sur (11.3.1)。我以为升级好系统之后，这台老旧的电脑可以恢复过来，结果却是彻底的黑屏。等我把电脑升级完成，时间已近中午，我彻底打消了去往健身房的计划。以及阅读《偷书贼》以及《地下室手记》的计划。

五一期间，我从前司节日赠送的积分，在积分商城购买了各种生活资料，其中包括一小袋子白粉，哦不，是白色的面粉。在购买白粉时，我已在想着拌面疙瘩汤。想起来，作为一个外乡人，我已经快有一年半没有回家乡的。我还记得我的家乡是以小麦为主食的，而我所在的这座烟雨的江南小城的主食当然是大米饭了。就这样，昨天三顿饭，我全部加入了面疙瘩，而且是不重样的玩法。想起来，一个人煮饭和吃饭有一个很大的好处，那就是可以天马行空，随心所欲地玩各种黑暗料理。只要自己想的到，只要自己不嫌麻烦，就可以随便下手。简单说一下昨晚的面汤做法：第一、将甘薯用菜刀刻成小小块状，并直接放入凉水的黑锅中；第二、将胡萝卜同样削成小小块置入锅中；第三、水开后，继续煮三五分钟；第四、把碗中拌好的面团打入开始中，并快速搅拌，将面团搅成面疙瘩；第五、锅开之后，打入一个鸡蛋……

五一过后，我在的这座江南小城已进入夏天模式。我想再过几天，就可以进入空调模式了。此时苹果电脑内置的蝶式键盘和触控板已经有些发烫之感。

此刻已经深夜十一点，我该休息了，有时间继续记录《偷书贼》的故事吧。

昨天晚上睡觉前看了一下天气预报，在可预见的一周内，将会是连绵不绝的雷雨天气。晚上睡觉前，通过网易云设置了故事FM最近一期《双向情感障碍》的单目播放。不知道怎么回事，定时的单曲播放设置失效，变成的单曲循环，并一直持续到天明。因此，我的睡眠并不是很深，在夜半时分，凌晨2点39分，隔着窗户听到了夜幕中中的第一声响彻夜空的惊雷，凌晨2点42分，夜空中响起了第三声雷鸣，随后可以清晰地听到窗外哗啦啦的雨声。尽管如此，但我也很快又再一次进入睡眠状态。

关于《偷书贼》的故事，其实是一个很简单的故事。在我看来，小说的故事比较适合小朋友们阅读，整体上并没有给我带来的什么震撼力。在被社会毒打已久的我，早已认识到人性的复杂。不过同偷书贼一样，读书对我来说同样是可以给我面对苦难生活的无穷的信念。假如生活欺骗了你（注：俄国的悲观主义诗人普希金），也许我们可以用文字对抗生活的苦痛和人生的虚无。

我已经写不下去了。其实，虽然是处于战争的灰暗时代，然而如今庸庸碌碌的我还是有些羡慕偷书贼的人生的。虽然生活很是困苦，然而偷书贼的生活却是充满了无穷的生命的活力。

此刻，坐在写字楼的格子间，望向黄浦江畔灯火绚丽的渡轮，还有滨江公路上来回穿梭的汽车，我的思绪犹如枯竭的河水，被慢慢地风蚀。

就这样吧。

想起来，今天计划读《地下室手记》的，可是看到这些意识过剩，充满精神分裂症的文字，太让人难以理解了。我还是想着换一些轻松一些的文字，缓解一下阅读的疲惫感。在我暂时放弃《地下室手记》，转向傅雷译的《名人传》时，又再一次被翻译给困住，语句读起来十分晦涩。我想我还是继续挑选一些其它的书来读一读吧，比如《生命不可承受之轻》或者其它的书。

告辞。
