河南烩面

08 / 25 / 2021 / 木

现在是北京时间2021年08月25日19时03分06秒。

大概在半个小时之前，我在小区附近不远处的，我常常去的一个面馆斥资13块人民币，吃了一份小碗牛肉烩面，以及8个左右的小蒜瓣。我们俗称的大蒜，为什么说是小蒜瓣，是因为在我拿蒜瓣的时候，在途径饭馆的水果店，耗资18.6元人民币采购了一把香蕉和一个红富士苹果。刚刚在带回来的笔记本电脑上处理了一件很小的工作上的事情。按我的个人想法，工作和生活的时间是要做到严格隔离的，不要有互相干扰和纠缠。不过，现在我是在消食，等会出去跑步，闲暇之余顺便处理一下，也算是解决了一份心事。

上周二的晚上在外出跑步的时候，在一个路口我看到有两个人在烧纸，我想这应该是烧纸钱之类的仪式吧。第二天，在我再次跑到这个路口的时候，还能看到有两摊灰烬。关于死亡，关于我自己的死亡，如果有一天在我死后，如果有人可以为我收尸的话，我希望可以在不要举行任何仪式，只需直接把尸体推进焚化炉。如果可以的话，我希望我的剩下的灰烬可以被抛向大海。当然，如果还有人记得我的话，还请不要有任何悲伤和遗憾。生命匆匆，人生一世，我只想希望可以静静地来，静静地走。我是一个不太喜欢热闹的人。喧嚣，热闹，只会让我感到烦躁和不安。

上周末读了一本技术书籍，大概内容是讲 Linux 操作系统和安全的。谈到计算机操作系统，我想多嘴几句。我们常常会听说什么编程，程序，代码以及码农，这些都是计算机系统的软件的范畴。我们也都知道在地球的另一边的，在大洋彼岸的西雅图有一个叫微软的科技公司，而微软的英文即 Microsoft。有软件，那么就有硬件。我们常见的硬件或者设备有可以直接观察和触摸到的显示器，鼠标，还有键盘，这些都是外围设备。多说一句，在我们这个慌乱的世界里，键盘还衍生出了一种叫做键盘侠的职业。

现代生活中，我们常见的是笔记本电脑，在笔记本电脑的键盘下面的壳子里则是板载着计算机的核心组件，如主板，中央处理器，内存，存储盘，以及网卡，显卡等等。在不是很早的过去，我们要知道还有一个叫机箱的黑盒子。随着集成电路的技术发展，计算机的组件都变得越来越小，小到可以手持和穿戴，即今天的智能手机，智能手环和手表。

说的简单一点，计算机的硬件就是一些电气组件或者逻辑电路。电路中有两个信号，可以简单的认为是高电压和低电压，分别可以用数字1和0表示。数字 1 和 0 就引出了数字中的二进制的概念。生活中，我们常常使用的是数字叫十进制，即逢十进一，而二进制则是逢二进一。在计算机系统或者软件工程中，除了二进制，常常用到的还有八进制，十六进制。

计算机系统的各种硬件设备的信号编码或者硬件资源调度的是就是上述中的计算机操作系统。我们常见的操作系统有个人电脑中的微软的 Windows，苹果的 macOS，移动端的如智能手机的 iOS，Android，智能手表中的苹果的 watchOS，智能电视中苹果的 tvOS，服务器端的 Debian, CentOS, Redhat, Ubuntu 等。

操作系统有一种叫做内核（Kernel）的东西，其中微软系的 Windows 是 Windows 内核，苹果系的如 macOS 则都是是 Darwin（达尔文），而其它的如 Android 以及常见的服务器操作系统的内核则都是 Linux 内核。

我们常用的智能手机有两种操作系统，其中一种苹果的 iPhone 的 iOS ，其它则都是基于 Android 的变种，包括但不限于如小米，华为，三星。在这个慌乱的世界里，最近爱国者手机也推出了爱国者操作系统，即鸿蒙操作系统，像是当年的汉芯一样，号称是国产操作系统。我们常用的个人电脑也有两种操作系统，其中一种是苹果的 macOS，另外一个就是微软的 Windows。

晚上九点，我要出去跑步了。晚安。

计算机名词解释：

  • 软件：Software

    • 相关的名词有代码（code），编码（coding），编程（programing），编程语言（programming language）
    • 程序或者代码编写的职业叫软件工程师（Software Engineer）即我们俗称的码农;

  • 硬件：Hardware

    • 计算：Central Processing Unit

      中文名叫中央处理器或处理器，简称 CPU，类似我们的大脑，简单来说，CPU 只做一件事情，即二进制的加法运算；

      • GPU: Graphical Processing Unit

        • 专用于图像处理的处理器，俗称显卡；

        • 在游戏或者视频剪辑领域用于图像渲染；在机器学习或者人工智能智能领域用于数据的处理；

    • 存储：Storage

      内存：英文名叫 Memory，是存储器的一种，也类似于我们大脑的记忆功能区；

      外存：类似我们的大脑的记忆空间是可贵而有限，所以计算机还有类似于竹简等大容量的存储设备，如硬盘，U盘；

    • 网络：Network

      如网卡（Network Interface Card），路由器（Router），光线（Fiber）；

    • 主板：Motherboard

      主板的作用用于集成把以上如处理器，内存，硬盘，网卡，还有外围设备的如鼠标，键盘以及移动硬盘等组件；
